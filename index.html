Introduction

This is an application that keeps track of and handles all of the information about consumers who have purchased services.

This project was created in order to complete one of the BTEC tasks. We were given the task of developing an application that would keep track of and manage all of the customers' information as well as the services they had purchased. This project relies entirely on the Liferay framework.

Technology used

Code technologies I got involved while working on this project:

  • Liferay Framework
  • SQL
  • Eclipse Maven
